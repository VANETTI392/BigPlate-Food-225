## 🍽️ BigPlate 225 - Menu Numérique

Menu numérique de restaurant pour plats ivoiriens avec affichage des prix en FCFA.

### ✨ Caractéristiques

- 🎯 **Interface intuitive et moderne** - Design responsive adaptable à tous les appareils
- 🍜 **15 plats ivoiriens** - Plats principaux, accompagnements, boissons et desserts
- 🛒 **Panier interactif** - Ajout/suppression d'articles, modification des quantités
- 🔍 **Recherche et filtrage** - Trouvez rapidement vos plats préférés
- 💾 **Sauvegarde locale** - Le panier persiste entre les sessions
- 📱 **Entièrement responsive** - Fonctionne sur desktop, tablette et mobile
- 💱 **Affichage en FCFA** - Prix en francs CFA ivoiriens

### 🚀 Démarrage rapide

1. Ouvrez `index.html` dans votre navigateur
2. Parcourez les plats disponibles
3. Cliquez sur un plat pour voir les détails
4. Ajoutez des articles à votre panier
5. Vérifiez votre commande et passez-la

### 🎨 Couleurs et Design

- **Couleur principale** : Orange (#E67E22) - Représente la chaleur et la cuisine
- **Couleur secondaire** : Rouge (#C0392B) - Accent et contraste
- **Couleur accent** : Vert (#27AE60) - Pour les actions positives (ajouter, commander)
- **Design moderne** : Dégradés, ombres et animations fluides

### 💻 Technologie

- **HTML5** - Structure sémantique
- **CSS3** - Grid, Flexbox, animations
- **JavaScript vanilla** - Pas de dépendances externes
- **LocalStorage** - Pour la persistance des données

### 🎯 Fonctionnalités principales

#### Recherche et Filtrage
- Recherche par nom ou description
- Filtrage par catégorie
- Mise à jour en temps réel

#### Gestion du Panier
- Ajout d'articles multiples
- Augmentation/diminution des quantités
- Suppression d'articles individuels
- Vider tout le panier

#### Détails des Plats
- Modal affichant les informations complètes
- Liste des ingrédients
- Sélection de quantité avant ajout

#### Confirmation de Commande
- Résumé de la commande
- Total calculé automatiquement
- Message de confirmation

### 📱 Responsive Design

L'application s'adapte à tous les écrans :
- **Desktop** : Grille multi-colonnes + barre latérale du panier
- **Tablette** : Grille à 2 colonnes + panier
- **Mobile** : Vue pleine largeur + panier modal
